# BOGO Offer Page 

This is a responsive React application showcasing a **Buy-One-Get-One (BOGO)** style offer section, built from a Figma design.

##  Features

- Three interactive offer boxes with expanding content
- Smooth transition animations on selection
- Dynamic size and color selectors
- **Add to Cart** functionality with:
  - Popup confirmation
  - Resetting of selections
- Fully responsive and centered layout

##  Tech Stack

- **React.js**
- **Tailwind CSS** (manual setup, no external UI libraries)
- **Vanilla JavaScript logic**
